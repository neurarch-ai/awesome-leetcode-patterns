# LeetCode 题型模式(中文版)

> 中文版。English: [patterns index](../../patterns/README.md)

这是本仓库 31 个题型模式的中文版本。每个模式一个文件,结构和英文版完全一致:
识别信号、核心思想、可背诵的代码模板(带时间/空间复杂度)、变体、经典题目表、
常见坑、以及延伸与相关模式。图和代码与英文版共用(代码是语言中性的),讲解正文
为简体中文。

参考页(解题框架、复杂度速查、术语表、题单、边界清单)目前仍是英文,位于仓库根
目录;本目录只翻译 31 个模式本身,这也是最常被反复阅读的部分。

每个模式文件的固定七段结构:

1. **识别信号** - 从题面认出这个模式:触发词、输入形态、关键约束。
2. **核心思想** - 机制是什么、为什么成立,以及相对暴力解省下的复杂度。
3. **模板** - 干净、可背诵的 Python(每个函数上方标了时间/空间复杂度)。
4. **变体** - 基础模板的子形态与常见变化。
5. **经典题目** - 训练该模式的 LeetCode 题目表(含难度与训练点)。
6. **常见坑** - 会让你解错的 bug 和边界情况。
7. **延伸与相关模式** - 面试官会如何深挖,以及相邻的模式。
